Parte 1: Teoría React
Práctica

¿Qué hace el hook useState?

El hook useState permite añadir y gestionar variables de estado local en componentes funcionales de React. Cuando el estado cambia, fuerza al componente a renderizarse de nuevo para reflejar la información actualizada en pantalla.

Un ejemplo sería un contador que cambia de valor cada que iteraciona en cumplimiento a una condición incrementandose o decreciendo.

¿Qué es un Fragmento (<> </>) y para qué sirve?

En el desarrollo web, el Fragmento <> </> (abreviatura de React.Fragment) es una sintaxis que permite agrupar múltiples elementos JSX en un componente sin añadir nodos adicionales o innecesarios al DOM real.

Ayuda a mantener el código ordenado.

¿Qué diferencia hay entre useContext y useState?

useState maneja datos locales dentro de un componente, es decir solo donde se declara.

useContext comparte datos entre varios componentes de forma global.

¿Cuál es la estructura básica de un componente funcional?

La estructura básica de un componente funcional (típicamente en bibliotecas como React) es una función de JavaScript o TypeScript que recibe parámetros de entrada y devuelve elementos visuales en formato JSX.

Estructura Esencial

Se compone de tres partes clave:

Importaciones: Carga las dependencias necesarias.

Declaración: El nombre del componente (debe comenzar con mayúscula) y los parámetros.

Retorno: El código JSX que define la interfaz de usuario.
